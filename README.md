# 📌 DataWorkbench – Smart Data Analytics Application
## 📖 Overview

DataWorkbench is an interactive web-based data analytics application that allows users to upload datasets and perform Exploratory Data Analysis (EDA) with powerful visualization tools.

The application simplifies data analysis by providing automated summaries, visualizations, and preprocessing tools.

## 🚀 Live Demo

👉 https://dataworkbench-app.streamlit.app/

## ✨ Features

1. Upload CSV and Excel files

2. Dataset preview

3. Missing value detection

4. Duplicate value detection

5. Statistical summary

6. Correlation heatmap

7. Interactive visualizations

8. Data filtering

9. Basic ML integration (if included)

10. Export processed data
